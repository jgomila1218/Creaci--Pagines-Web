Creació de pagines web

Hem creat aquesta pagina web per poder demostrar la capacitat de la IA per a la generació de webs. La IA utilitzada ha estat kimi i ens ha aportat un codig html amb el cual hem publicat la web.

El promt utilitzat per a la web ha estat el seguent: Quiero hacer una pagina web. Crea un codigo HTML que tenga un titulo de la web. Mas abajo debe haber tres subapartados con un titulo y texto para cada uno. El sub apartado uno contiene 3 imagenes y el subapartado dos contiene dos videos.
